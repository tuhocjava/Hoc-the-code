Học thẻ code trong HTML

Trong bài trước chúng ta đã biết cách hiển thị một đoạn code lên trình duyệt bằng cách sử dụng HTML Entities. Nhưng nếu đoạn code của chúng ta quá dài chúng ta không thể sử dụng HTML Entities một cách thủ công vì mất quá nhiều thời gian, để khắc phục chúng ta có thể sử dụng thẻ code.

Ví dụ sau hiển thị nội dung của thẻ head bằng cách sử dụng thẻ code:

Đầu tiên truy cập trang web: https://codebeautify.org/html-escape-unescape,
sau đó copy đoạn code của bạn cần hiển thị trên trang web của bạn rồi paste vào khung HTML Escape/Unescape và chọn Escape, tiếp đó copy đoạn mã trong khung The HTML result Data và paste vào thẻ code trong project 

<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link rel="stylesheet" href="style.css">
    <title>Bài 8: Học về thẻ code</title>
</head>

Lưu ý: khi sử dụng thẻ code chúng ta nên kết hợp với thẻ pre để giữ nguyên định dạng code. Thẻ pre và thẻ code nên viết trên một dòng và paste nội dung vào giữa
